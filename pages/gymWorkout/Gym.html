Gym Workout

This workout is planned according to gym facilities.
These would include use of weights, bands, machines etc.

Sr. Day Game
1 Monday Biceps
2 Tuesday Triceps
3 Wednesday Back
4 Thursday Chest
5 Friday Legs
6 Saturday Cardio
7 Sunday Rest

Biceps

  1. Seated DB Curl - 2 warm-ups, 3 sets, 10-15 reps
  2. Cable Bar Curl - 1 warm-ups, 3 sets, 10-12 reps
  3. Seated EZBar Curl - 3 sets, 1 drop set, 8-10 reps

Triceps

  1. Cable Extension - 2 warm-ups, 3sets, (last drop set) 12-15 reps
  2. Incline Skullcrusher - 2 warmup, 3set, 6-8 reps
  3. Cable Bar Pushdown - 3-4 sets, (fail last set), 8-10 reps

Back

  1. Close grip Lat Pull Down, 4 set 15-20 reps
  2. Seated cable rows, 4 set 15 reps
  3. One arm dumbell row, 3 set 15 reps each

Chest

  1. Incline Dumbell press, 4 set 15 reps
  2. Bench Press, 4 set 15 reps
  3. Cable crossover, 4 sets 15-20 reps

Legs

  1. Romanian Deadlift, 3 set 15 reps
  2. Leg Extensions, $ set 15-20 reps
  3. Barbell Squat, 4 set 15 reps
  4. Reverse leg curls, 4 set 15 reps
  5. Calf raises, 4 set 25 reps

Cardio

  1. 2 mile running, 13 minutes
  2. 100 pushups, 4 set 25 reps
  3. 50 squats, 2 sets 25 reps
  4. 50 crunches, 2 sets 25 reps
  5. 20 pullups, 4 sets 5 reps
